Implementación de un ETL a la medida

En este proyecto se implementa un proyecto capaz de tomar datos csv y llevarlos a una base de datos MySQL, puede soportar archivos de grandes tamaños, se implementa logging, manejo de errores y resumir en caso de errores utilizando procesamiento por lotes, submódulos de git, expresiones regulares para extracción de valores de los datos entre otras cosas (dockerización algunos tests).
